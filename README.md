# 🌤 Weather App — React + Vite + Tailwind CSS

A real-time Weather Application built with React, Vite, and Tailwind CSS during my MERN Stack Internship at SVVV, Indore.

## ✨ Features
- Search any city by name
- Shows current temperature in Celsius
- Humidity, wind speed, and pressure info
- Weather condition with icon
- Loading spinner while fetching data
- Error message for invalid city names
- Responsive glassmorphism UI

## 🛠️ Tech Stack
- React.js
- Vite
- Tailwind CSS
- OpenWeatherMap API
- JavaScript Fetch API (async/await)

## 🧠 React Concepts Used
- useState — weather data, loading, error states
- async/await — fetching data from API
- Conditional rendering — spinner, error, weather card
- Props — passing data to child components
